Arduino Radar System

📌 Title
Arduino Radar System Using Ultrasonic Sensor and Servo Motor

A radar-inspired object detection system that uses an ultrasonic sensor mounted on a servo motor to scan the surrounding area and measure the distance of nearby objects. The system continuously rotates the sensor, detects obstacles, and provides real-time distance measurements.

---

## 🛠 Technologies

### Hardware
- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- SG90 Servo Motor
- Breadboard
- Jumper Wires
- USB Cable

### Software
- Arduino IDE
- Embedded C/C++
- Serial Monitor

---

## ✨ Features

- Real-time object detection
- Distance measurement using ultrasonic waves
- 180° scanning using servo motor
- Continuous radar-like sweeping motion
- Serial Monitor output for distance visualization
- Low-cost and beginner-friendly implementation
- Easy to modify and expand for advanced applications

## ⚙️ The Process

### 1. Hardware Setup
Connected the HC-SR04 ultrasonic sensor to the Arduino Uno and mounted it on a servo motor to enable rotational scanning.

### 2. Servo Configuration
Programmed the servo motor to rotate from 0° to 180° and back continuously, allowing the sensor to scan a wide area.

### 3. Distance Measurement
Used the ultrasonic sensor to emit sound waves and calculate the distance to nearby objects based on the time taken for the echo to return.

### 4. Data Processing
Collected distance measurements at different servo positions and displayed the information through serial communication.

### 5. Testing and Calibration
Performed multiple tests to verify distance accuracy and ensure smooth servo movement.

### 6. Final Implementation
Integrated all components into a working radar system capable of detecting and measuring objects in real time.

2. Connect the servo motor to the Arduino.
3. Open the project in Arduino IDE.
4. Select the correct Arduino board and COM port.
5. Upload the code to the Arduino.
6. Open the Serial Monitor.
